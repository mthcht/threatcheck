Configure which threat intel services appear when you select text.

No ads No data collection Open source It just works

How it works

Select a single IOC for a quick lookup popup, or select an entire paragraph from a threat report to extract all IOCs at once. Defanged IOCs like 192[.]168[.]1[.]1 or hxxps://evil[.]com are auto-refanged.

Services marked WORKS function out of the box. Services marked NEEDS SETUP only appear once configured.

Reload open tabs after installing or updating.
