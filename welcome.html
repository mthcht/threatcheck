1 How it works

Select any text on any web page. ThreatCheck automatically detects IOCs and shows a popup with relevant lookup services. No copy-pasting between tabs.

Select a single IOC

↓ popup appears with VirusTotal, AbuseIPDB, Shodan, Spur, GreyNoise...

Select an entire paragraph

↓ bulk panel extracts all 3 IOCs with auto-deduplication and refanging

3 Key features

  • Auto-refanging - handles defanged IOCs like hxxps://evil[.]com and admin[at]evil[.]com
  • Defang copy - one-click copy IOCs in safe format for reports and tickets
  • Bulk extraction - select a paragraph from a threat report, all IOCs extracted and deduplicated
  • URL + domain splitting - selecting a URL automatically offers both the full URL and the extracted domain
  • API auto-enrichment - scores and context from configured services appear directly in the popup
  • Expandable context panels - click any score badge to see full details without leaving the page
  • 1,285 TLDs - full IANA TLD list for accurate domain detection, no false positives
