ThreatCheck

Select any indicator on a web page - or highlight an entire paragraph from a threat report, blog post, paste, or ticket - and instantly extract and look up every IOC across multiple threat intel platforms. Supports defanged indicators, bare URLs, and bulk extraction with auto-deduplication.

Supported IOC types
IPv4 / IPv6 Domain URL Hash (MD5/SHA1/SHA256) Email CVE Event ID Error Code
